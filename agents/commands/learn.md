# Learn Command

Use this command when new publishable knowledge should enter the context layer.

## Input

- New idea, research finding, decision, or implementation result
- Optional source material

## Procedure

1. Search existing notes for overlap.
2. Decide whether to update an existing note or create a new note.
3. If source material is private, rewrite and generalize it before using it.
4. Add wikilinks to adjacent notes.
5. Run required checks.

## Output

- Created or updated note path
- Reason for placement
- Checks run

## Completion Check

No raw private context enters `notes/`.
