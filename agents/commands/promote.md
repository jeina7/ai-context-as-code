# Promote Command

Use this command when a draft or worklog should become durable context.

## Procedure

1. Identify the reusable idea.
2. Remove incidental implementation chatter.
3. Choose the correct note type.
4. Link to decisions, patterns, or project notes.
5. Run required checks.

## Output

- Promoted note path
- Source material used
- Checks run

## Completion Check

The promoted note should remain useful without knowing the original session.
