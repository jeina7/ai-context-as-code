# Synthesize Command

Use this command to turn several notes into a clearer durable note.

## Input

- A set of related notes or a topic

## Procedure

1. Read the relevant notes.
2. Identify overlap, contradiction, and missing structure.
3. Create or update a synthesis note.
4. Preserve links to source notes.
5. Avoid summarizing private source material directly.
6. Run required checks.

## Output

- Synthesis note path
- Source notes used
- Changes made

## Completion Check

The synthesis should reduce future reading effort without erasing important distinctions.
