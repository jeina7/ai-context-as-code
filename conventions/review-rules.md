# Review Rules

Before a note is published, check:

- Is it publishable?
- Does it duplicate an existing note?
- Does it have required frontmatter?
- Are wikilinks valid?
- Does the summary explain the note quickly?
- Is the content durable enough to be useful later?

For AI-generated changes, also check:

- The agent did not invent sources.
- The agent did not expose private context.
- The agent did not turn a specific private event into publishable content without generalizing it.
