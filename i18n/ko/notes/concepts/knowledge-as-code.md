# Knowledge as Code

## 요약

Knowledge as Code는 오래 남길 노트를 소프트웨어 프로젝트처럼 다루는 방식이에요.
노트는 버전 관리 안에 있고, 규칙을 따르며, 배포 전에 검증을 통과해요.
이렇게 하면 사람과 에이전트가 지식을 더 쉽게 재사용할 수 있어요.

## 패턴

평범한 파일을 기준 원본으로 두고, 그 위에서 색인과 화면을 생성해요.

```text
markdown notes
→ metadata build
→ validation
→ static site
→ agent review
```
