# 스스로 진화하는 context loop

## 요약

유용한 context system은 사용을 통해 좋아져요.
agent나 사람이 빠진 규칙, 낡은 노트, 반복되는 correction을 발견하면 durable context가 업데이트되어야 해요.
이 loop는 노트뿐 아니라 노트를 만드는 규칙도 개선해야 해요.

## loop

```text
use context
→ find gap
→ propose update
→ validate update
→ commit improved note, rule, or script
```

## 규칙

반복 feedback은 chat history에만 남으면 안 돼요.
convention, validation rule, durable note 중 하나가 되어야 해요.
