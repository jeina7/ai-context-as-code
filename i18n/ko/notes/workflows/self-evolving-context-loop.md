# 스스로 진화하는 맥락 loop

## 요약

유용한 맥락 시스템은 사용을 통해 좋아져요.
에이전트나 사람이 빠진 규칙, 낡은 노트, 반복되는 correction을 발견하면 durable 맥락이 업데이트되어야 해요.
이 loop는 노트뿐 아니라 노트를 만드는 규칙도 개선해야 해요.

## loop

```text
use context
→ find gap
→ propose update
→ validate update
→ commit improved note, rule, or script
```

## 규칙

반복 feedback은 채팅 기록에만 남으면 안 돼요.
규칙, validation 규칙, 오래 남길 노트 중 하나가 되어야 해요.
