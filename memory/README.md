# Memory Pointers

This folder holds compact pointers into durable context.
It is not a dump of private memory.

## Rules

- Prefer links to notes over copied summaries.
- Keep only stable, reusable facts.
- Do not store private personal, financial, company, or relationship details.
- Update the linked durable note when a pointer becomes stale.

## Current Pointers

- `product-context.md`: why this repository exists and what it should become.
- `agent-config.md`: how agents should find rules, commands, skills, and memory.
