# Agent Config Pointer

Agent configuration should use familiar surfaces instead of a custom runtime model.

Durable source notes and files:

- `AGENTS.md`
- `CLAUDE.md`
- `agents/commands/`
- `skills/`
- `memory/`
- `conventions/agent-rules.md`
- `notes/projects/agent-runtime-references.md`

Use these files before changing how agents read or update this repository.
