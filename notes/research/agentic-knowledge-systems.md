# Agentic Knowledge Systems

## Summary

Agentic knowledge systems help AI agents work with durable context instead of starting from scratch each session.
The current trend is moving from isolated prompts toward context engineering, codified context, and agent-maintained knowledge layers.
acac.sh sits in that direction as a lightweight, markdown-first implementation.

## Observations

Relevant industry terms:

- context engineering
- agentic context engineering
- codified context
- knowledge layer
- agent harness
- Model Context Protocol

These terms point to the same underlying problem: AI agents need structured, validated, evolving context to act reliably.

## Boundary

This project starts as a context layer, not a full agent harness. Harness capabilities such as tool permissions, execution traces, and automatic commits can come after the context source is trustworthy.
