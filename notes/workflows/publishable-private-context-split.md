# Publishable Private Context Split

## Summary

A publishable knowledge system should not be a direct copy of a private vault.
Private notes can contain raw context, but publishable notes need review and generalization.
The split lets a person keep rich private memory while publishing reusable knowledge safely.

## Pattern

Use two spaces:

- private source space for raw, sensitive, and personal material
- publishable memory space for reviewed, generalized, reusable notes

Material moves from private to publishable only through a review step.

```text
private source
→ private staging
→ safety review
→ rewrite for publishing
→ publishable notes
```
