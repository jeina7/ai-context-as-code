# Skills

This folder holds reusable agent procedures.
The files here are not product features by themselves.
They are stable procedures that an agent can follow or later package as tool-native skills.

## Current Skill Candidates

- `context-review.md`: review notes, links, safety, and generated metadata.
- `publishable-import.md`: turn private-source candidates into safe, generalized notes.
- `interface-polish.md`: review UI alignment, density, and responsive behavior.

## Rules

- A skill should describe when to use it, inputs, steps, and verification.
- A skill should point to durable conventions instead of duplicating every rule.
- A skill must not require private vault access unless the caller explicitly provides it.
